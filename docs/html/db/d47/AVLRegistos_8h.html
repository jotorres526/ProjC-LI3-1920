Detailed Description

Definicão do tipo AVLRegistas e das funcoes que a manipulam É a base do catálogo das filiais e da faturacao A diferenca para a AVL é que aqui guarda tambem um registo alem do nodo No caso de ser usado para faturacao o registo é um RegistoFaturacao No caso de ser usado para filial o registo é uma ListaRegistosFilial
